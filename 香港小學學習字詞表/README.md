# 香港小學學習字詞表（可瀏覽版）

教育局《香港小學學習字詞表》全部 **3,129** 字及其相關詞語，整理成方便在 GitHub 上
直接翻閱的 Markdown 表格。資料由 [`../word-crawler`](../word-crawler) 的抓取工具從
教育局[網上查詢系統](https://www.edbchinese.hk/lexlist_ch/)取得並匯出 CSV，再由
本目錄的 [`to_markdown.py`](to_markdown.py) 轉成下列 Markdown。

## 內容

| 檔案 | 說明 |
| --- | --- |
| **[`characters.md`](characters.md)** | 全部 3,129 字，按總筆畫分節，檔首附目錄可跳轉。每列含部首、普通話拼音、粵拼、學習階段、相關詞語數目，及該字全部相關詞語。 |
| **[`words/`](words/)** | 相關詞語（連逐詞普通話拼音／粵拼／學習階段），按字的筆畫分檔（`stroke-01.md` … `stroke-32.md`），另有 [`words/README.md`](words/README.md) 目錄。 |

> 為何 `words/` 要拆檔：GitHub 對過大的檔案會改顯示原始碼而不渲染。故按筆畫拆成
> 多個小檔（最大約 90 KB），避免把 17,704 列塞進單一表格。

## 欄位

**`characters.md`**

| 欄位 | 說明 |
| --- | --- |
| 字 | 繁體字 |
| id | 系統內部編號 |
| 部首 | 部首（已去「部」字，如「一」）|
| 普通話拼音 | 多讀音以「；」分隔 |
| 粵拼 | 粵語拼音；多讀音以「；」分隔，〔〕內為同音字提示 |
| 學習階段 | 第一 / 第二 |
| 詞數 | 相關詞語數量 |
| 相關詞語 | 全部相關詞語，以「、」分隔 |

**`words/stroke-NN.md`**

| 欄位 | 說明 |
| --- | --- |
| 字 | 所屬單字 |
| 相關詞語 | 詞語 |
| 普通話拼音 | 該詞普通話拼音 |
| 粵拼 | 該詞粵拼 |
| 學習階段 | 該詞所屬階段 |

> 小技巧：在 GitHub 檔案頁按 <kbd>t</kbd>／<kbd>/</kbd> 或瀏覽器 <kbd>Ctrl/⌘</kbd>+<kbd>F</kbd>
> 可快速搜尋。

## 統計（由 CSV 實際計算）

- 《香港小學學習字詞表》：**3,129** 字
  - 第一學習階段：**1,921** 字
  - 第二學習階段：**1,208** 字
  - 其中 **239** 字只有單字條目、沒有相關詞語
- 相關詞語：**17,704** 條（每字每詞一列）；去重後不同詞語約 **8,317** 條
  （同一詞語常見於多個組成字之下，例如「三文治」同時見於「三」「文」「治」）

> 全部字級欄位（部首、筆畫、普通話拼音、粵拼、學習階段）在 3,129 字皆完整無缺。
> 個別詞語（如「水庫」）在查詢系統中同時列於第一及第二學習階段，本工具照實保留。

## 重新產生

原始資料在 [`../word-crawler/out/`](../word-crawler/out)（`characters.csv`、`words.csv`）。
更新 CSV 後，於版本庫根目錄執行即可重建本目錄的 Markdown：

```bash
uv run 香港小學學習字詞表/to_markdown.py
# 預設讀 ../word-crawler/out，寫回本目錄；可用 --csv-dir / --md-dir 覆寫
```

抓取流程、判別方法與資料來源等詳情，見 [`../word-crawler/README.md`](../word-crawler/README.md)。
